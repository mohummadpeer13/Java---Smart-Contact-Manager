Gestionnaire de Contact Intelligent est une application web complète permettant de gérer efficacement une base de contacts avec des fonctionnalités avancées. Elle offre un système d'authentification sécurisé, la gestion des profils utilisateurs avec photos, et un système de notifications asynchrone.

Fonctionnalités principales

Gestion des contacts

Création, modification, suppression et recherche avancée de contacts avec catégorisation.

Authentification avancée

Connexion classique et via Google OAuth2 avec gestion des rôles et permissions.

Gestion des profils

Téléchargement et stockage cloud des photos de profil avec interface intuitive.

Système de notifications

Envoi d'emails asynchrones via architecture événementielle avec Apache Kafka.
